About the Application

Creator Content Sync is a locally-hosted desktop utility built with Streamlit. It is designed specifically for independent creators to manage their original video portfolio and sync content to their TikTok profile via the official TikTok Content Posting API.

This tool allows users to:

  • Select original video files from local storage.
  • Edit titles, descriptions, and metadata locally.
  • Preview content before publishing.
  • Directly upload and publish to their authenticated TikTok account.

Note: This application is for private, local use and is not a public-facing SaaS platform.

Privacy Policy

Last Updated: April 2024

1. Data Processing

The App operates locally on the user's hardware. It does not collect, store, or transmit user data to any external servers owned by the developer.

2. TikTok API Usage

The App uses the following TikTok API scopes:

  • user.info.basic: To display the profile name of the logged-in creator.
  • video.upload: To securely transfer the user's selected video files to TikTok's servers.

3. Data Security

All authentication tokens are stored locally on the user's machine. The developer has no access to these tokens or the user's content.
